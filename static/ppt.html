策略模式

BaseLLMClient 定义统一接口，各模型独立实现。ModelService 工厂方法动态创建

应用工厂模式

create_app() 统一创建 Flask 实例，根据 FLASK_ENV 加载不同配置

注册表模式

ToolService._tools 字典管理工具，新增只需注册，不改调度逻辑

核心功能：对话流程

从用户输入到 AI 流式回复的完整链路

1. 用户打字/语音 → POST /api/v1/chat/stream 2. Flask → ChatService.prepare_chat_context() 加载角色 System Prompt 从 MySQL 加载历史 (滑动窗口10轮) 组装: [System, 历史..., User新消息] 3. ModelService 获取模型客户端 4. DeepSeek/Qwen 流式返回 json.dumps 编码每个 chunk 5. SSE → marked.js 实时 Markdown 渲染 6. 完成: 持久化MySQL + Token统计 + TTS朗读

JSON-SSE 换行保护

chunk 用 json.dumps 编码，前端 JSON.parse 解码。换行符不再被 split('\\n') 吃掉，Markdown 代码块语法完整

流式 Markdown

每个 chunk 调 marked.parse() 实时渲染，完成后 hljs 高亮。CDN 未加载时内置正则兜底

停止生成

AbortController + fetch signal。用户点停止 → abort() 中断流，已输出内容保留

模型参数面板

Temperature / MaxTokens / Top-P 滑块实时调节，覆盖默认值，localStorage 持久化

语音功能

3 引擎 11 音色，浏览器语音输入 + 服务端语音合成

🎤 语音输入 (STT)

浏览器 SpeechRecognition API，免费，continuous=false 说完自动停止，实时填入输入框并自动发送。Ctrl+M 快捷键。配有麦克风诊断工具页面

🔊 语音输出 — 火山引擎 (8 音色)

小源(BV700)/大夏(BV701)/清心(BV001)/知行(BV002)/知性(BV004)/彤彤(BV005)/小艾(BV402)/大庆(BV401)。Bearer Token 认证，POST 字节跳动 API，返回 base64 MP3 → 前端 Blob 播放

🔊 语音输出 — 腾讯云 (3 音色)

智瑜(1001)/智美(1003)/智聆精品(1050)。tencentcloud-sdk-python-tts SDK，SecretId/Key 认证，VoiceType 指定音色

发送前清洗 + 失败兜底

cleanForTTS() 去除 Markdown 语法(代码块/粗体/链接/标题)，只保留纯朗读文字。腾讯云限 150 字自动截断。API 失败自动切浏览器 speechSynthesis

数据库设计

MySQL 8.0 utf8mb4, ORM 操作, 4 张核心表

conversations (会话表) id VARCHAR(36) PK / title VARCHAR(200) / user_id INDEX model VARCHAR(32) / role_id VARCHAR(36) / created_at / updated_at messages (消息表) id INT PK AUTO_INC / conversation_id FK / role VARCHAR(16) content TEXT / tokens INT / created_at roles (角色表) id VARCHAR(36) PK / name VARCHAR(100) / prompt TEXT icon VARCHAR(10) / is_builtin BOOLEAN / created_at / updated_at users (用户表) id VARCHAR(64) PK / api_key UNIQUE / quota / created_at

ORM 映射

SQLAlchemy db.Model 定义表结构，Python 操作数据库无需手写 SQL。首次启动 db.create_all() 自动建表

Token 估算

中文字符 x1.5 + 英文单词 x0.3 + 数字 x0.5 + 其他 x1.0。每条消息创建时自动计算

硬删除机制

删除会话时先批量 DELETE messages WHERE conversation_id=X，再 DELETE conversation，彻底清理
