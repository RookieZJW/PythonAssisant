# 📖 使用指南

Python 多功能智能助手 (v2.6) 的安装、配置和日常使用教程。

---

## 一、安装

### 1.1 环境要求

- Python ≥ 3.11
- MySQL 8.0（需运行中）
- （可选）Redis 7.0

### 1.2 安装步骤

```bash
# 1. 进入项目目录
cd E:\PythonAssistant

# 2. 安装依赖
pip install -r requirements.txt

# 3. 配置 MySQL 数据库
# 确保 MySQL 服务正在运行，项目首次启动会自动创建数据库和表

# 4. 配置环境变量
# 编辑 .env 文件，至少填入 DeepSeek API Key：
DEEPSEEK_API_KEY=sk-你的密钥
```

### 1.3 启动

```bash
python run.py
```

启动后浏览器打开 **http://localhost:5000**，会自动跳转到登录页。

---

## 二、注册和登录

### 2.1 注册账号

1. 打开 http://localhost:5000/login
2. 点击 "没有账号？立即注册"
3. 输入用户名（至少2位）和密码（至少6位）
4. 点击注册，自动跳转到聊天页面

### 2.2 登录

1. 在登录页输入用户名和密码
2. 点击登录

> ⚠ 手机验证码登录、微信/QQ扫码登录功能UI已就绪，但因需要企业营业执照认证，暂不可用。

### 2.3 退出登录

聊天界面左侧边栏点击 **"退出"** 按钮。

### 2.4 数据隔离

每个用户登录后只能看到：
- 自己的聊天记录
- 自己创建的角色（内置角色所有人共享）
- 自己的背景历史

---

## 三、基本功能

### 3.1 开始对话

1. 在底部输入框输入消息
2. 按 **Enter** 发送（Shift+Enter 换行）
3. AI 回复会逐字流式显示

### 3.2 停止生成

AI 输出过程中，发送按钮变为红色 **⏹**，点击即可停止。

### 3.3 切换模型

点击顶栏 **🔄 模型** 按钮，在 DeepSeek 和通义千问之间切换。

### 3.4 切换角色

顶部下拉框选择不同角色（编程老师、数据分析师等），AI 的回复风格会随之改变。

---

## 四、角色管理

### 4.1 创建角色

1. 点击顶栏 **＋ 角色**
2. 选择图标 → 填写名称和提示词 → 保存
3. 角色名称不可与已有角色重复

### 4.2 编辑/删除角色

- 选择角色后，下拉框右侧出现 **✏️** 和 **🗑️** 按钮
- 所有角色（含内置）都可编辑和删除（至少保留一个）

---

## 五、主题设置

### 5.1 切换预设主题

1. 点击顶栏 **🎨 主题**
2. 点击预设主题卡片（共12套）

### 5.2 自定义调节

在主题面板中可调节：
- 面板透明度（侧边栏/顶栏/输入区）
- 聊天气泡透明度
- 按钮透明度
- 下拉框透明度
- 输入框透明度
- 边框粗细和颜色
- 气泡阴影

### 5.3 自定义背景

1. 在"自定义背景"输入框中粘贴图片或视频 URL
2. 选择类型（图片/视频）和显示模式（全屏/仅聊天区）
3. 点"应用"
4. 支持本地文件路径（如 `D:\图片\bg.jpg`），系统自动代理
5. 点"💾 存入历史"保存，下次可直接点击历史记录切换

---

## 六、语音功能

### 6.1 语音输入

1. 点击输入框旁的 **🎤** 按钮开始说话
2. 说完自动停止识别并发送
3. 快捷键：**Ctrl+M**
4. 麦克风测试：http://localhost:5000/static/mic-test.html

### 6.2 语音输出 (TTS)

1. AI 回复完成后自动朗读（需勾选"自动朗读"）
2. 底部下拉框切换音色（火山引擎8种 + 腾讯3种）
3. 点击 **🔊 朗读** 手动朗读最后一条回复
4. 点击 **⏹ 停止** 停止朗读

---

## 七、文件上传

1. 点击输入框旁的 **📎** 按钮
2. 选择文件（支持 PDF/Word/TXT 及常见代码文件）
3. 按钮变为 **📄** 绿色，表示附件已就绪
4. 在输入框中基于附件提问
5. 再次点击按钮可移除附件

---

## 八、模型参数调节

1. 点击顶栏 **⚙️ 参数**
2. 拖动滑块调节：
   - **Temperature**：0=严谨，2=创意
   - **Max Tokens**：回复最大长度
   - **Top-P**：词汇多样性

---

## 九、对话管理

### 9.1 查看历史

左侧边栏显示所有历史对话，点击切换。

### 9.2 重命名

鼠标悬停到对话上 → 点击 **✏️** → 输入新标题。

### 9.3 删除

鼠标悬停 → 点击 **✕** → 确认删除（会同时删除数据库中的消息）。

### 9.4 导出

点击顶栏 **📥 导出** → 自动下载 Markdown 文件。

## 十、配置第三方服务（可选）

### 10.1 语音合成 TTS

**火山引擎**（免费额度）：
1. 前往 https://console.volcengine.com/ 注册
2. 开通语音合成服务
3. 获取 APP_ID 和 Access Token
4. 填入 `.env`：`VOLCANO_TTS_APP_ID` 和 `VOLCANO_TTS_TOKEN`

**腾讯云 TTS**（每月100万字符免费）：
1. 前往 https://console.cloud.tencent.com/ 注册
2. 开通语音合成
3. 获取 SecretId 和 SecretKey
4. 填入 `.env`：`TENCENT_TTS_SECRET_ID` 和 `TENCENT_TTS_SECRET_KEY`

### 10.2 限流（Redis）

安装 Redis 7.0 并启动，系统自动接入。无 Redis 时限流功能自动跳过。

## 十一、常见问题

| 问题 | 解决 |
|------|------|
| 启动报 MySQL 错误 | 检查 MySQL 是否运行，.env 密码是否正确 |
| AI 不回复 | 检查 API Key 是否正确；访问 /api/v1/health |
| 流式输出一次性返回 | Nginx 需 proxy_buffering off |
| 语音识别不工作 | 先测 /static/mic-test.html；检查浏览器麦克风权限 |
| 自定义背景不显示 | 类型选"视频"时模式选"全屏显示" |
| Markdown 渲染异常 | Ctrl+F5 强制刷新 |
